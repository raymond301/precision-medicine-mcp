# Getting Started

Quick start guides for new users of the Precision Medicine MCP system.

## Contents

- **[Installation Guide](installation.md)**
  - **Claude Code (CLI)** — clone, explore, and run tests from your terminal (recommended for study groups)
  - **Claude Desktop (GUI)** — visual interface for running patient analyses
  - Usage examples and troubleshooting
- **[Claude Desktop Configuration Files](desktop-configs/README.md)**
  - Pre-built and template config files (JSON)
  - IMPORTANT: you will have to edit file paths for your environment setup

## Which Setup Should I Use?

| You Are... | Use This | Time |
|------------|----------|------|
| Student in a study group | [Claude Code (CLI)](installation.md#quick-start-claude-code) | 5 min |
| Developer exploring the code | [Claude Code (CLI)](installation.md#quick-start-claude-code) | 5 min |
| Clinician running analyses | [Claude Desktop (GUI)](installation.md#quick-start-claude-desktop) | 10 min |
| Instructor setting up a class | [Educator Guide](../for-educators/README.md) | 30 min |

## Prerequisites

- **Python 3.11+** and **git**
- **[uv](https://docs.astral.sh/uv/)** — Python package manager (replaces pip/venv)
- **Claude Code:** also requires Node.js 18+ (`npm install -g @anthropic-ai/claude-code`)
- **Claude Desktop:** download from [claude.ai/download](https://claude.ai/download)
