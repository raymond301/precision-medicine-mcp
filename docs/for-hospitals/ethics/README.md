# Ethics & Bias Framework for Precision Medicine AI

This directory contains a comprehensive framework for detecting and mitigating algorithmic bias in precision medicine AI/ML workflows, aligned with FDA AI/ML SaMD guidance, AMA Code of Medical Ethics Opinion 2.3.2, and NIH All of Us diversity requirements.

## Contents

- **[ETHICS_AND_BIAS.md](ETHICS_AND_BIAS.md)** — Complete methodology: data representation analysis, fairness metrics (demographic parity, equalized odds, calibration), proxy feature detection, and ancestry-aware confidence scoring.
- **[BIAS_AUDIT_GUIDE.md](BIAS_AUDIT_GUIDE.md)** — Practical step-by-step audit checklist (3-5 days per audit) with a PatientOne demonstration showing the framework applied to an ovarian cancer workflow.
