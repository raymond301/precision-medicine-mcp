## Procedure

1. **Notify Users:** Send maintenance email 7 days in advance.
2. **Rotate Keys:** Rotate service account keys and API secrets.
3. **Patch Images:** Build new container images with latest security base.
4. **Verify:** Run full integration regression suite.
