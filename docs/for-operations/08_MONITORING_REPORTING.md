## 8. Monitoring and Reporting

### 8.1 Observability Layers

1. **Infrastructure:** Cloud Run metrics, Cloud Monitoring.
2. **AI Behavior:** Langfuse (prompt replay), Datadog LLM Observability.
3. **Application:** Traces, error rates (Sentry/Cloud Logging).
4. **Dependency:** Anthropic API health, Epic FHIR availability.

### 8.2 Reporting Cadence

- **Weekly:** Operational metrics review (IT Operations).
- **Monthly:** SLA compliance report (IT Manager → IT Director).
- **Quarterly:** Comprehensive SLA review (Change Advisory Board).
